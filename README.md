# ReservaServiciosComunitarios
Proyecto de Reserva de Servicios Comunitarios
```
## Descripción
El proyecto consiste en una aplicación web que permita a los usuarios reservar servicios comunitarios, como por ejemplo, canchas de fútbol, salones de fiesta, entre otros. La aplicación debe permitir a los usuarios registrarse, buscar servicios, reservarlos y pagarlos. Además, debe permitir a los administradores de los servicios, gestionar la disponibilidad de los mismos, así como la información de los usuarios que han reservado.
